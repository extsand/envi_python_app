Environment Example
This app allows showing environment variables from application settings. For example, you should add your value to the environment.

Value naming is important part of this application. See code sample bellow:

{# export EXAMPLE_ENV="https://db.com:3442"#} export APP_USERNAME=HollyMolly
export APP_DB=https://db.com:3442
export APP_PASSWORD=bestpasswordever
Application output:
{% for item in data_send %}
{{ item }}
{% endfor %}
